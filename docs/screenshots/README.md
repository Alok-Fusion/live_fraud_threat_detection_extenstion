# Screenshots

Place UI screenshots in this folder so they can be referenced from the main [`README.md`](/README.md).

This folder currently contains lightweight **SVG placeholders** so the main README renders cleanly.

To use real screenshots:

- Replace the placeholders with your own images (recommended: `*.png`).
- Update the image links in [`README.md`](/README.md) if you change extensions.

Suggested screenshot file names:

- `popup.svg` (extension popup)
- `popup-url-check.svg` (URL Check result in popup)
- `dashboard-overview.svg` (dashboard overview section)
- `dashboard-live-feed.svg` (dashboard live feed section)
